This project introduced an efficient machine learning-based detection system for identifying Kidney Stones. This model accurately categorized a diverse dataset of 1700 images into specific classes. Leveraging OpenCV for image handling, have applied techniques like median blur and thresholding for image enhancement. CNN model's parameters were fine-tuned through Random Search and ensured model reusability. This model exhibited an impressive accuracy of approximately 90% on test data.
